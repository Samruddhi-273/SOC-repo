1.What is Machine Learning?
Machine Learning is a branch of AI.
Computers learn patterns from previous data.
ML helps systems improve automatically with experience.

2.Types of Machine Learning
1.Supervised Learning:Uses labeled data.
The model learns from inputs and correct outputs.
Example:Predicting coin currency using weight and size.
2.Unsupervised Learning:Uses unlabeled data.
The computer finds hidden patterns or groups.
Example:Grouping cricket players based on performance.
3.Reinforcement Learning:Learning through rewards and feedback.
The system improves after correcting mistakes.
Example:Image recognition improving after feedback.

3.How Machine Learning Works
Basic process:
1.Input data is given
2.Model learns patterns
3.Predictions are made
4.Feedback improves accuracy
