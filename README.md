# Summer of Code (SOC) 2026

This repository documents my learning journey through the Summer of Code (SOC) program. Over eight weeks, I explored Python programming, Machine Learning, Deep Learning, Computer Vision, Large Language Models (LLMs), Exploratory Data Analysis, and Multimodal AI. The program combined theoretical concepts with hands-on projects, culminating in the development of an AI-powered Industrial Quality Assurance System.

## Week 7 & 8 - LLM Integration and Deployment

Integrated Computer Vision with Large Language Models to develop an end-to-end AI-powered Industrial Quality Assurance System.

### Topics Covered

* Streamlit Web Application Development
* YOLOv8 Model Integration
* Local LLM Deployment using Ollama
* Llama 3.2 Integration
* AI-powered Inspection Report Generation
* User Interface Design
* End-to-End Multimodal AI Pipeline
* Application Deployment

### Technologies Used

* Streamlit
* Ultralytics YOLOv8
* Ollama
* Llama 3.2
* Pandas
* Pillow

### Project Workflow

1. Upload a steel surface image.
2. Detect surface defects using the trained YOLOv8 model.
3. Display the detected image with bounding boxes.
4. Summarize detected defects and confidence scores.
5. Generate a professional inspection report using Llama 3.2.
6. Download the AI-generated inspection report.

### Key Outcome

Built and deployed a complete Multimodal AI application that combines Computer Vision and Large Language Models to automate industrial quality inspection and reporting through an interactive Streamlit interface.
