# Summer of Code (SOC) 2026

This repository documents my learning journey through the Summer of Code (SOC) program. Over four weeks, I explored the foundations of Python programming, Machine Learning, Deep Learning, and Large Language Models (LLMs), gaining both theoretical understanding and practical implementation experience.

---

## Week 1 - Python Fundamentals

Built a strong foundation in Python programming by learning:

* Variables, Data Types, and Operators
* Conditional Statements and Loops
* Functions and Modular Programming
* Lists, Tuples, Dictionaries, and Sets
* File Handling and Exception Handling
* Object-Oriented Programming (OOP)

### Key Outcome

Developed the ability to write structured Python programs and solve basic computational problems independently.

---

## Week 2 - Machine Learning

Explored the core concepts of Machine Learning and the complete model development workflow.

### Topics Covered

* Introduction to Machine Learning
* Data Preprocessing
* Train-Test Split
* Linear Regression
* Logistic Regression
* K-Nearest Neighbors (KNN)
* Decision Trees
* Model Evaluation Metrics
* Overfitting and Underfitting

### Tools & Libraries

* NumPy
* Pandas

### Key Outcome

Learned how to prepare datasets, train machine learning models, and evaluate their performance.
